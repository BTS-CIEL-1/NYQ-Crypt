Comment ça fonctionne

NYQ Crypt utilise un algorithme de cryptage symétrique personnalisé pour sécuriser vos données.

  1. Entrez une clé de cryptage robuste (elle sera nécessaire pour décrypter)
  2. Saisissez du texte ou chargez un fichier
  3. Cliquez sur "Crypter" pour sécuriser vos données
  4. Pour décrypter, fournissez la même clé et le texte crypté

Attention : Si vous perdez votre clé, vos données ne pourront pas être récupérées!
