Comment ça fonctionne

NYQ Crypt utilise un algorithme de chiffrement symétrique personnalisé pour sécuriser vos données.

  1. Entrez une clé robuste (elle sera nécessaire pour déchiffrer)
  2. Saisissez du texte ou chargez un fichier
  3. Cliquez sur "Chiffrer" pour sécuriser vos données
  4. Pour Déchiffrer, fournissez la même clé et le texte chiffré

Attention : Si vous perdez votre clé, vos données ne pourront pas être récupérées!
